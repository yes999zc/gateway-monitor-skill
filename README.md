# Gateway Monitor macOS Skill

OpenClaw Gateway 监控技能（macOS），内置 LaunchAgent + watchdog + 一键安装脚本。

## 快速上手（3 条命令）

```bash
# 1) 安装
bash scripts/install.sh

# 2) 查看状态
bash scripts/status.sh

# 3) 卸载
bash scripts/uninstall.sh
```
